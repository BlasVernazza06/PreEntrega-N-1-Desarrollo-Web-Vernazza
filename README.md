# 🏫 Página Web Colegio San Martín

Este repositorio contiene el desarrollo de la **landing page** del colegio **San Martín**, un sitio web pensado para ofrecer una presentación moderna, clara y accesible de la institución educativa.

## 🌐 Descripción del proyecto

El objetivo principal de esta página es brindar información institucional del colegio San Martín a padres, alumnos y comunidad en general. La estructura está centrada en un diseño limpio y adaptable, con secciones pensadas para destacar los valores, servicios y novedades del establecimiento.

## ⚙️ Stack Tecnológico

El desarrollo se ha realizado utilizando tecnologías web básicas, enfocadas en la práctica y dominio de estilos modernos:

- **HTML5**: para la estructura semántica del sitio.
- **CSS3**: para los estilos visuales, incluyendo:
  - **Grid Layout** y **Flexbox** para un diseño responsivo y flexible.
  - **Pseudoclases** para mejorar la experiencia de usuario con efectos y transiciones.
- **Git**: para el control de versiones local.
- **GitHub**: como plataforma para alojar el código y colaborar en línea.
